# TDU s2j-Shikkou

This is the website and portfolio of TDU Shikkou, 
which was developed using Nuxt and NotionAPI.


## Techs

  - Nuxt
  - Notion API

## PackageVersion
- `nuxt 2.15.7`
-  `@nuxtjs/axios ^5.13.6`
-  `bootstrap-vue ^2.21.2`

## Installation

### Setup

#### Install the depedencies

```bash
npm install
```

#### Start the project

```bash
# for dev
npm run dev
```
