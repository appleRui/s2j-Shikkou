# TDU s2j-Shikkou

This is the website and portfolio of TDU Shikkou, 
which was developed using Nuxt and NotionAPI.

## Link
[東京電機大学　二部執行委員会 HP](https://tdu-s2j-shikkou.netlify.app/)

## Techs
 - Front
  - Nuxt
 - API
  - Notion API

## Package version
- `nuxt 2.15.7`
-  `@nuxtjs/axios ^5.13.6`
-  `bootstrap-vue ^2.21.2`

## Installation

### Setup

#### Install the depedencies

```bash
npm install
```

#### Start the project

```bash
# for dev
npm run dev
```
